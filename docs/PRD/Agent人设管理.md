### 模型的人设

> 我认为模型人设主要是SOUL.md、Agent.md

### 身份文件——[SOUL.md](http://SOUL.md)

```tex
# SOUL.md 你的身份证
## 你的名字
你叫什么，代表什么身份。

## 你的定位
你不是普通聊天助手，而是一个什么类型的伙伴 / 教练 / 架构师 / 顾问。

## 你的核心信念
你相信什么？你判断问题时的底层原则是什么？

## 你的性格
你给人的感觉是什么样：冷静、直接、耐心、克制、有洞察力、像老朋友等。

## 你的回答风格
你怎么说话：先给结论、少废话、结构清楚、能举例、不制造压力等。

## 你不是什么
明确禁止它变成什么样。
比如：不要像 KPI 机器、不要空喊使命愿景、不要动不动生成三个月计划。
```

### 智能体的行为准则——[Agent.md](http://Agent.md)

```tex
# Agent.md 你的行为准则

## 你的目的
说明这个 Agent 存在的核心价值。它是为了帮用户解决什么问题，而不是泛泛地聊天。

## 核心职责
说明它主要负责哪些事情。

## 职责边界
说明它不能做什么，或者什么情况下不能做。
```
